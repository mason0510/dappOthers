买方用户须知：

1、请在“广告”规定的时间内完成付款，并点击确认付款按钮。买方主动取消订单或者订单超时取消，系统将记录买方取消订单一次。

2、买方当日累计3笔取消的，会限制当日买入功能。

3、发布求购广告，当天累计取消订单达到3笔的，系统会限制您当天的买入功能。

4、因为卖方未提供有效支付方式，导致系统记录，买方取消订单，买方已因此被限制当天买入功能的，可以联系平台官方，要求解除限制。

5、请勿在未完成付款或不付款的情况下点击“我已付款”，该行为是属于恶意行为。当此类订单发生申诉，卖方可拒绝成交，情节严重的，系统将冻结买方账户。

6、您在10分钟内未付款且不回应卖方的，当此订单出现申诉时，卖方可拒绝成交。

7、请选择您本人的实名支付账户（如银行账户、支付宝账户、微信支付账户）来付款。如果您使用非实名账户或他人账户付款，当此订单出现申诉时，卖方可选择退款不成交。

8、为了保证交易的及时性，请选择实时到账的汇款方式，例如支付宝、微信支付、银行实时汇款等。为确保到款的及时性，节假日或工作日的17：00-9：00，超过5万以上的汇款请分批支付，请勿使用支付宝转银行卡、微信转银行卡方式支付。

9、买方点击“我已付款”两个小时后，卖方未收到付款，订单出现申诉时，卖方可以拒绝成交。

10、请核对卖方最新收款方式，确认收款账户准确性，如买方未按照订单生成的约定账户汇款，将自行承担资金安全风险。

11、进行中的订单对应的数字货币资产会锁定在平台；如您转账完成，并点击“我已付款”5分钟后，卖方未放币，则您可选择申诉，如您完全符合以上操作，我们会判定资产属于您。

12、如遇到付款账号实名和平台认证实名不一致，卖方有权选择以下操作：

  • 要求买方和付款方手持身份证或护照等，进行视频KYC，证明资金来源合法；
  • 原路退款不成交；
  • 扣押数字货币72小时不放行。72小时后，由卖方决定放行或者原路退款。

注：买方在转账的汇款备注里，备注六位数字的付款参考号即可。

该“付款参考号”仅用于标识该笔汇款是对应于该笔订单，买方的汇款按约定到账的，视为已遵守交易规则。

请勿填写类似BTC、比特币、火币等关键字，否则卖方可要求退款不成交。

 

卖方用户须知：

1、当您在卖出时，请认真确认您所卖出的价格，因广告价格问题产生的申诉，如买方无违规操作，我们会判定资产属于买方并直接划转。

2、买方点击“我已付款”5分钟后未收到账，卖方可选择申诉；如买方不遵循先付款后点击付款、付款不能在两小时以内到达账户或付款后订单取消，卖方可选择申诉，并退款不成交。

3、收款时请仔细核实，买方的付款账户实名信息与平台实名信息是否一致。如不一致，请注意收款风险，当此类订单发生申诉时，卖方可选择退款拒绝成交。因用户接受非实名付款，导致他人的支付账户被冻结的，平台会调查问题资金来源，并且有权直接冻结该用户平台账户、取消商家认证资格。

4、请在登录网银查看收款无误后，第一时间放行。买方无任何违规，买方标记订单状态为“已付款”后的20分钟以内，卖方无法完成放行的，当此订单发生申诉时，买方有权要求退款不成交，如卖方拒绝协调，我们会直接放行给买方并冻结卖方账户。

5、为了确保交易的及时性，请发布广告时务必确保能在线并及时处理订单；如您无法保证及时处理交易订单，请及时下架，以免造成不必要的申诉纠纷。

6、如遇到付款账号实名和平台认证实名不一致，卖方有权选择以下操作：

  • 要求买方和付款方手持身份证或护照等，进行视频KYC，证明资金来源合法；
  • 原路退款不成交；
  • 扣押数字货币72小时不放行。72小时后，由卖方决定放行或者原路退款。

注：对于交易纠纷，平台有最终裁判权。对于恶意冻结订单、恶意付款、利用交易涉嫌诈骗等行为，平台可选择冻结账户，必要时协助司法机关进行处理。

 

（广告主）用户须知：

1、成为商家才能发布广告。

2、如您不在设备旁或离线无法及时处理订单，请提前下架广告。此类广告如产生订单，属于正常订单，按正常交易流程处理。

3、发布求购广告时，当天累计取消订单达到3笔的，系统会限制您当天的买入功能。

4、商家收款时，请仔细核实买方的付款账户实名信息与买方的平台实名信息是否一致。

因商家接受非实名付款，导致他人的支付账户被冻结的，平台会调查问题资金来源，并且有权直接冻结该商家平台账户、取消商家认证资格。

交易区买入时需完成实名认证。
